Kuusamo

Kuusamo is an open source learning management system (LMS) written in PHP.

features

  • Create courses

    Build courses, modules and lessons.

  • JSON blocks

    Lessons are saved as API-friendly blocks of JSON (behind the UI).

  • Media types

    Supports text, video, audio, images and downloads.

  • Certificate generation

    Tracks progress and automatically generates certificates.

  • Plugins

    Plugin architecture allows you to extend Kuusamo.

  • Open source

    Extend and modify Kuusamo as you wish.

getting started

Kuusamo installs via Composer. See the docs for full installation instructions.

composer create-project kuusamo/project

docs

Documentation can be found in its own git repository. It is written in Markdown so is readable offline. If you have any questions or find something that is not working, you can raise an issue on Github.
